# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities by emailing us directly rather than creating public issues. We will acknowledge receipt within 48 hours and provide an estimated timeline for resolution.
